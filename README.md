# WarEra Pulse

**Real-time market & conflict terminal for [warera.io](https://app.warera.io).**

A Bloomberg/TradingView-style dashboard that turns the game's economy and wars into a
single live terminal: resource prices, order books, candle charts, equipment prices,
active battles and the nations on the rise — all updating in real time.

🔗 **Live:** https://www.warera-pulse.info

## Features

- **Markets** — the 21 tradeable resources with live prices and sparklines, plus the
  military equipment tiers (weapons & armor) with current prices.
- **Price panel** — candlestick chart (1-week / 1-month views) with SMA/EMA, 24h
  high/low/open, a live order-book depth ladder and a one-click jump to trade in-game.
- **Active conflicts** — ongoing battles with ground points, tick countdown and damage
  share, plus a direct link into each fight.
- **Hot nations** — the weekly damage leaderboard.
- **Global feed** — wars, battles, alliances, region changes and more, as they happen.

Everything refreshes live and the whole thing fits one screen — built to keep open on a
second monitor while you play.

## Built with

Next.js · React · TypeScript · Tailwind CSS · lightweight-charts.
